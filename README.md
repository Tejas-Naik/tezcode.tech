# TezCode.tech - Python Programming Course Website

A modern, responsive website for TezCode.tech's Python programming course, built with React.js and Tailwind CSS.

## Features

- Modern, responsive design
- Interactive course curriculum
- Project showcase
- Student testimonials
- Pricing plans
- FAQ section
- Contact form
- Google Calendar integration for consultations

## Tech Stack

- React.js
- Tailwind CSS
- React Scroll
- React Icons
- Google Calendar API

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/tezcode.tech.git
cd tezcode.tech
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Build for production:
```bash
npm run build
```

## Project Structure

```
src/
  ├── components/     # Reusable components
  ├── sections/       # Page sections
  ├── constants/      # Constants and data
  ├── assets/         # Images and other assets
  └── styles/         # Global styles
```
